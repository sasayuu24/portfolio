skill

HTML ★★★★☆

基本的な文書構造を理解し、セマンティックなマークアップを意識したコーディングができます。 レスポンシブ対応を想定したレイアウト構築が可能です。

CSS ★★★★☆

Flexbox・擬似クラスを用いたレイアウト調整ができます。 デザインカンプを再現するためのスタイリングやアニメーション実装も行えます。

jQuery ★★★☆☆

基本的なDOM操作・イベント処理を用いて動きのあるUIを実装できます。 既存コードのカスタマイズやプラグインを活用した処理が可能です。

JavaScript ★★★☆☆

変数・関数・配列などの基礎構文を理解しています。 簡単な動的処理やインタラクションの追加が可能です。

XD ★★★☆☆

ワイヤーフレーム作成からデザインカンプ制作まで一貫して対応できます。 リンク設定による簡易プロトタイプ作成も可能です。

Photoshop ★★★★☆

写真の切り抜き・補正・合成などの基本操作ができます。 バナー制作やWebデザインに必要な画像加工が対応可能です。

Illustrator ★★★☆☆

ロゴやシンプルなイラスト制作を行うことができます。 ベクターデータの調整やパス操作など基本操作を習得しています。

Excel ★★★★☆

MOS資格を独学で取得。基礎関数の使用や表作成などの正確な操作が可能です。 データの整理・集計や、見やすい資料になるようなレイアウト調整にも対応できます。

Word ★★★★☆

MOS資格を独学で取得。文書作成の基本操作を習得しています。 見出し設定・レイアウト調整など、整った資料作りができます。

Power point ★☆☆☆☆

スライド構成を考え、分かりやすい資料を作成できます。 アニメーション設定や図解などを用いて伝わるデザインが可能です。

Figma ★☆☆☆☆

FigmaでワイヤーフレームやUIデザイン、簡易プロトタイプの制作が可能です。 使いやすさを意識した設計を心掛けています。

Canva ★★☆☆☆

CanvaでSNS画像やバナーを制作出来ます。テンプレートを活かしつつ、配色や文字組みにこだわった見やすいデザインが可能です。

ibis Paint ★★★★☆

イラストや簡単な加筆修正が可能です。ブラシ調整やレイヤー操作を活用し、スムーズに使用できます。

Premiere Pro ★☆☆☆☆

カット編集、テロップ制作、BGM、音声調整など、素材を整理して分かりやすくまとめる編集が出来ます。

about

茨城県出身。 趣味は、人の感情や行動の仕組みを観察・分析することです。
自分の機嫌の傾向を分析するのが特に好きです。

高校卒業後、 工場勤務を経て、アイドル活動ではグループの運営や企画立案、メンバーとの関係づくりに携わりました。グループ全体をまとめながら、 SNSを通じて発信やファンとの交流を行う中で、企画・運営・チームマネジメントの楽しさを学びました。その後は営業事務として、社内メンバーがスムーズに動けるようサポートや教育、情報共有の仕組みづくりを担当。その後は職業訓練校（Web ディレクター科）で、Web制作の工程やデザイン・コーディングの基礎を学びました。制作全体の流れを把握する中で、 各工程をつなぐディレクションの重要性に惹かれました。
